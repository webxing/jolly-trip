# Jolly Trip - 视觉设计规范

## 1. 色彩系统

### 1.1 主色调
- 主要品牌色：#4A90E2（天空蓝）
  - 代表：探索、自由、梦想
- 辅助色：#50E3C2（青碧）
  - 代表：自然、生命、活力
- 点缀色：#F5A623（金橙）
  - 代表：温暖、友好、能量

### 1.2 场景配色
- 自然场景：以绿色和蓝色为主
- 城市场景：以灰色和褐色为主
- 文化场景：根据地区特色定制
- 节日场景：鲜艳而富有节日气氛

## 2. 界面设计

### 2.1 布局原则
- 简洁清晰
- 层次分明
- 重点突出
- 操作便捷

### 2.2 交互元素
- 按钮设计：圆角矩形，带有微弱阴影
- 图标设计：简约现代，保持一致性
- 菜单设计：层级清晰，易于导航
- 提示设计：醒目但不打扰游戏体验

## 3. 角色设计

### 3.1 玩家角色
- 多样化的外观选择
- 符合人体工程学的比例
- 丰富的表情系统
- 细节精致的服装系统

### 3.2 NPC设计
- 具有地域特色
- 性格鲜明
- 动作自然
- 服饰考究

## 4. 场景设计

### 4.1 建筑风格
- 保持地域特色
- 注重细节刻画
- 合理的比例关系
- 优美的建筑线条

### 4.2 自然环境
- 真实的地理特征
- 丰富的植被系统
- 动态的天气效果
- 自然的光影变化

## 5. 特效设计

### 5.1 环境特效
- 天气变化效果
- 自然光影效果
- 水面反射效果
- 植被动态效果

### 5.2 交互特效
- 技能施放效果
- 成就达成效果
- 任务完成效果
- 社交互动效果

## 6. 动画设计

### 6.1 角色动画
- 自然的行走/跑步动作
- 流畅的交互动作
- 丰富的表情动画
- 真实的服装物理效果

### 6.2 场景动画
- 建筑物细节动画
- 环境物件互动
- 群众NPC行为
- 节日庆典效果

## 7. UI设计规范

### 7.1 字体使用
- 主标题：Bold 24px
- 副标题：SemiBold 18px
- 正文：Regular 16px
- 说明文字：Light 14px

### 7.2 图标设计
- 线条粗细统一
- 视觉重量平衡
- 风格统一简约
- 含义清晰明确
